# Chews


## Purpose: 

This mobile app was developed to help users find directions to restaurants in their area.
## Tech Used:

<ul>
<li>React Native</li>
<li>Node.js</li>
<li>Express.js</li>
<li>Two Google Maps APIs</li>
<li>The Yelp API</li>
<li>PostgreSQL</li>
